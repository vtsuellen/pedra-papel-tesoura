# Pedra, Papel e Tesoura
Este é um simples jogo de Pedra, Papel e Tesoura desenvolvido em Python. O jogo permite que o jogador escolha entre Pedra, Papel ou Tesoura, e então o computador faz uma escolha aleatória. 

## Como Jogar
- Execute o script em um ambiente Python.
- Escolha uma das opções de jogada: Pedra, Papel ou Tesoura.
- O computador fará uma escolha aleatória.
- O resultado da rodada será exibido.
